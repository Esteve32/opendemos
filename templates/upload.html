Batch Upload Instructions
Method 1: Command Line Interface
python scripts/gemini_manager.py batch-upload --directory /path/to/your/html/files
Method 2: Copy to Export Directory
  1. Copy all your HTML files to the html_exports directory
  2. Go to the Files page
  3. Click "Batch Upload" to upload all files to Google Drive
Method 3: Using the API

For programmatic access, use the REST API endpoints:

  • POST /api/batch_upload_to_drive - Batch upload to Google Drive
  • GET /api/search?q=query - Search files
  • POST /api/cleanup_duplicates - Remove duplicates
